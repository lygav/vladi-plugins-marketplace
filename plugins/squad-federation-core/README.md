# squad-federation-core

Manage multiple permanent AI teams from a single leadership squad — each with its own worktree, agents, and accumulated knowledge.

[![Documentation](https://img.shields.io/badge/docs-lygav.github.io-blue)](https://lygav.github.io/vladi-plugins-marketplace/)

> Built on [Squad](https://github.com/bradygaster/squad). This plugin adds the federation layer for multi-team orchestration.

## Install

```bash
copilot plugin install squad-federation-core@vladi-plugins-marketplace
```

## Key Features

- **Conversational setup** — say *"federate this project"* and a wizard configures everything
- **Persistent worktrees** — each team gets its own git branch, agents, and knowledge
- **Three-layer architecture** — Core → Archetype → Project, each owning its config
- **Signal protocol & knowledge lifecycle** — inter-team communication, learning flows, and cross-team pattern detection
- **OTel monitoring** — built-in Aspire dashboard with traces, metrics, and events

## Getting Started

```
> federate this project
> @federation onboard a new team
> launch the payments team
```

See the [full documentation](https://lygav.github.io/vladi-plugins-marketplace/) for the complete setup guide, architecture details, and API reference.
